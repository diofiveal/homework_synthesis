D-latch (D - защёлка) - одна из простейших последовательных схем, пропускающий сигнал по переднему фронту тактового сигнала CLK.
D-защёлка построена с помощью RS-триггера. RS-триггер имеет 2 входа: R (reset) и S (set) и 2 выхода Q и !Q. RS-триггер построен с помощью двух элементов ИЛИ-НЕ.
